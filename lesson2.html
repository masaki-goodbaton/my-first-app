Section 1

HTML の基礎

📄 HTMLの基本構造

すべてのHTMLファイルはこの構造から始まります。

HTML
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>ページタイトル</title>
</head>
<body>
  <!-- ここにコンテンツを書く -->
</body>
</html>

🏷️ よく使うHTMLタグ

  • <h1> ~ <h6> - 見出し（h1が最も大きい）
  • <p> - 段落（テキストのブロック）
  • <a href="URL"> - リンク
  • <img src="" alt=""> - 画像
  • <ul> / <ol> - リスト（箇条書き / 番号付き）
  • <div> - ブロック要素のグループ化
  • <span> - インライン要素のグループ化

🔗 ブロック要素 vs インライン要素

ブロック要素は横幅いっぱいに広がり、前後に改行が入ります。

インライン要素はコンテンツの幅だけを占め、横に並びます。

HTML
<!-- ブロック要素 -->
<div>これはブロック要素です</div>
<p>これもブロック要素です</p>

<!-- インライン要素 -->
<span>これは</span><span>インライン要素です</span>
Section 2

CSS の基礎

🎨 CSSの書き方

CSSはセレクタ、プロパティ、値の3つで構成されます。

CSS
/* セレクタ { プロパティ: 値; } */

h1 {
  color: #333;
  font-size: 24px;
  font-weight: bold;
}

.class-name {
  background-color: #f0f0f0;
  padding: 16px;
  border-radius: 8px;
}

#id-name {
  margin: 0 auto;
}

📏 よく使うCSSプロパティ

  • color - 文字色
  • background-color - 背景色
  • font-size - 文字サイズ
  • margin - 外側の余白
  • padding - 内側の余白
  • border - 枠線
  • width / height - 幅 / 高さ
  • display - 表示形式（block, flex, grid）
💡
margin vs padding：marginは要素の「外側」、paddingは要素の「内側」の余白です。border（枠線）を基準に考えると覚えやすいです。
Section 3

レイアウトの基本

📦 Flexbox

要素を横並び・縦並びに配置する強力なレイアウト手法です。

CSS
.container {
  display: flex;
  justify-content: center;   /* 横方向の配置 */
  align-items: center;      /* 縦方向の配置 */
  gap: 16px;               /* 要素間の隙間 */
}

/* よく使う justify-content の値 */
/* center       → 中央揃え */
/* space-between → 両端揃え */
/* flex-start   → 左揃え */
/* flex-end     → 右揃え */

🖥️ レスポンシブデザイン

画面サイズに応じてレイアウトを変えるためにメディアクエリを使います。

CSS
/* 基本スタイル（モバイルファースト） */
.container {
  width: 100%;
  padding: 16px;
}

/* タブレット以上 */
@media (min-width: 768px) {
  .container {
    max-width: 720px;
    margin: 0 auto;
  }
}
Section 4

便利な Tips

✅
DevToolsを活用しよう：ブラウザでF12キーを押すと開発者ツールが開きます。要素の検証やCSSのリアルタイム編集ができます。
✅
HTMLの入れ子ルール：<p>タグの中に<div>は入れられません。ブロック要素の中にインライン要素を入れるのが基本です。
⚠️
classとidの使い分け：classは複数の要素に使えますが、idはページ内で1つだけ。基本的にはclassを使いましょう。

⌨️ 覚えておきたいショートカット

  • Ctrl + S - ファイルを保存
  • Ctrl + Z - 元に戻す
  • Ctrl + Shift + I - DevToolsを開く
  • ! + Tab - HTMLの雛形を自動生成（VS Code）
Section 5

よくあるエラーと対処法

❌
閉じタグ忘れ：<div>を開いたら必ず</div>で閉じましょう。インデントを揃えると見つけやすくなります。
❌
CSSが反映されない：セレクタ名のスペルミス、セミコロン;の付け忘れ、ファイルのリンク切れを確認しましょう。
❌
画像が表示されない：srcのパスが正しいか、ファイル名の大文字・小文字が合っているか確認しましょう。
